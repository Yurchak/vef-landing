# SEO Checklist for Landing Pages

Use this checklist to ensure that generated landing page structures and content are optimized for search engines.

## 1. On-Page Basics
- [ ] **Primary Keyword:** Identified and included in H1, Title Tag, and first 100 words.
- [ ] **Title Tag:** Compelling, under 60 characters, includes primary keyword.
- [ ] **Meta Description:** Action-oriented, under 160 characters, includes variation of keyword.
- [ ] **URL Structure:** Clean, descriptive, includes keyword (e.g., `/service-name` not `/p=123`).

## 2. Heading Hierarchy
- [ ] **H1:** Only one H1 per page. Describes the core offer.
- [ ] **H2s:** Break up main sections (Features, Benefits, Testimonials). Include secondary keywords.
- [ ] **H3s:** Use for detailed subsections (specific feature breakdowns).

## 3. Content Optimization
- [ ] **Keyword Density:** Natural flow; avoid keyword stuffing.
- [ ] **LSI Keywords:** Use semantically related terms (e.g., if page is about "coffee makers," use "brewing," "filter," "barista").
- [ ] **Readability:** Short paragraphs (1-3 lines), bullet points, bold text for emphasis.
- [ ] **User Intent:** Does the content answer *what* the user is looking for? (Informational, Transactional, Navigational).

## 4. Technical & UX (Content-Side)
- [ ] **Alt Text:** Descriptive text for all image placeholders.
- [ ] **Internal Linking:** Plan for links to other relevant pages (e.g., "See our Pricing," "Read the Blog").
- [ ] **CTA Clarity:** Button text should be descriptive (e.g., "Get Your Free Quote" vs. "Submit").
