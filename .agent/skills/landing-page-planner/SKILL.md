# Landing Page Planner

## When to use this skill
- When the user wants to create a new landing page or website.
- When the user asks for help with "information architecture" or "content strategy" for a page.
- When the user wants to audit or improve an existing page's structure.
- When the user provides a "brain dump" of business info and wants a structured output.

## Workflow
1.  **Discovery & Context Gathering**:
    -   Adopt the persona of a world-class SEO/CRO/Copywriting expert.
    -   Interview the user to gather missing critical information (USP, Target Audience, Goal, Competitors).
    -   *Crucial:* Do not proceed to generation until you have a solid understanding of *who* this is for and *what* they should do.
2.  **Strategy & Tone Selection**:
    -   Based on the discovery, propose a **Tone of Voice** (e.g., Professional, Playful, Urgent, Luxury).
    -   Select a **Conversion Framework** (AIDA, PAS, FAB) that fits the industry/offer.
3.  **Drafting the Plan**:
    -   Create a structural outline (H1, H2, Section purpose).
    -   Map out the content flow.
4.  **Content Generation**:
    -   Write the actual copy for each section, following the chosen framework and SEO best practices.
    -   Include suggestions for visuals/media and layout.

## Instructions

### 1. The Interview (Phase 1)
Start by acknowledging the user's request and asking targeted questions. Do not ask *everything* if they already provided it. Key data points needed:
-   **Product/Service:** What are we selling?
-   **Target Audience:** Who is the ideal customer? What is their pain point?
-   **Unique Selling Proposition (USP):** Why us and not the competitor?
-   **Goal:** What is the primary conversion action? (Sign up, Buy, Call)

### 2. The Plan (Phase 2)
Present a "Page Strategy" before writing full copy.
-   **Structure:** List the sections (Hero, Problem, Solution, Social Proof, CTA).
-   **Rationale:** Explain *why* this structure works for their specific audience.

### 3. The Content (Phase 3)
When generating the full content:
-   **Headings:** Write punchy, benefit-driven H1s and H2s.
-   **Body Copy:** Use the selected tone. Keep it readable.
-   **SEO:** Integrate the primary keyword and related terms naturally. (Refer to `references/seo-checklist.md`).
-   **Visuals:** Describe the ideal imagery for each section (e.g., "Image of happy customer using product").
